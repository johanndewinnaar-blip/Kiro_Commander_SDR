# Kiro Hooks — Commander SDR

This folder contains hook creation prompts and guardrail documentation. Do not rely on hooks as authority. Hooks must enforce Commander authority, not replace it.

Recommended initial hooks are documented as natural-language hook definitions because Kiro hook configuration should be created and reviewed inside Kiro before being enabled.
