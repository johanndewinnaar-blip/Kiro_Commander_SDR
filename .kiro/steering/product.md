# Product Steering — Commander SDR

Commander SDR is a Seiertech SaaS platform for Security Drift & Response within a Security Command and Control category.

Do not describe Commander SDR as an MVP, generic dashboard, SIEM, SOAR, CTEM-only tool, SOC triage platform or incident-response replacement.

The product must include from day one: Command Centre, case management, vulnerability management, exposure management, asset intelligence, identity intelligence, control coverage, architecture/topology, Security C2, OODA views, Direction Boards, connector framework, mock connectors, tenant admin, editable baselines, Commander AI, governance/reporting, CISO dashboard, Phase 2 real connector readiness and Phase 3 pilot hardening.
